# Interactive Choropleth Maps with Plotly: A Complete Tutorial

This repository contains an honors option project that teaches how to build an
interactive choropleth map using **Plotly** and a publicly available dataset of
global corporate greenhouse gas emissions (2022–2023) from Kaggle.

The tutorial walks through the full workflow:

- Loading and exploring a dataset  
- Cleaning and preparing country-level data  
- Converting country names to ISO-3 codes for mapping  
- Building an interactive choropleth map with Plotly  
- Customizing color scales, hover labels, and layout  
- Exporting the final visualization as an HTML file  
- Hosting the visualization using GitHub Pages  

---

## Why This Tutorial?

Plotly’s documentation includes basic choropleth examples, but it does **not**
provide a complete, beginner-friendly workflow that covers:

- preparing a real dataset  
- handling inconsistent country names  
- converting to ISO codes  
- exporting interactive HTML  
- hosting the visualization online  

This tutorial fills that gap by providing a clear, step-by-step guide using a
real-world dataset.

---

## Dataset

**Global Corporate GHG Emissions (2022–2023)**  
Source: Kaggle  
Link: https://www.kaggle.com/datasets/alitaqishah/global-corporate-ghg-emissions-20222023

The dataset includes greenhouse gas emissions reported by corporations around
the world. For this tutorial, emissions are aggregated by country to create a
global choropleth map.
