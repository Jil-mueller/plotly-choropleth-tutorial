Interactive Choropleth Maps with Plotly

This project is an honors option tutorial demonstrating how to build an interactive choropleth map using Plotly and a publicly available dataset of global corporate greenhouse gas emissions (2022–2023).

The tutorial walks through:

  • Loading and cleaning a real dataset
  • Converting country names to ISO codes
  • Building an interactive choropleth map
  • Styling and customizing the visualization
  • Exporting the final map as HTML
  • Hosting the visualization using GitHub Pages
